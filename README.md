## About This Repository

This repository contains an interactive HTML figure used to explore how changes in the parameters alpha12 and alpha21 affect the modeled interaction between viral variants.

The figure is intended as a reproducible and shareable complement to scientific work, allowing readers to interact with the model output instead of only viewing a static image.

## Included Figure

### Figure 4h: Exploration of alpha12 and alpha21

Interactive exploration for the AcMNPV-LV cooperative desert model.

The figure can be opened directly in a web browser through GitHub Pages or by downloading the repository and opening `index.html`.
